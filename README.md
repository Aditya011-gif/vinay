# Ashoba Powermatrix Pvt. Ltd. Website

This is a modern, minimal, premium corporate website built for **Ashoba Powermatrix Pvt. Ltd.** using **React + Vite**. The design is conversion-focused and optimized for both B2B and B2C electrical manufacturing clients.

---

## 🛠️ Key File Locations

- **Central Configuration**: [config.js](file:///c:/Users/adity/Downloads/vinay/src/config.js) — Contains all editable fields (company name, phone number, WhatsApp link, Google Maps embed URL, product details, services lists, etc.).
- **Global Stylesheet**: [index.css](file:///c:/Users/adity/Downloads/vinay/src/index.css) — Custom variables for the color palette (`#0F3D91` primary, `#F4A300` accent), premium typography (Inter), smooth scroll configurations, and slider animations.
- **Main Landing Page**: [App.jsx](file:///c:/Users/adity/Downloads/vinay/src/App.jsx) — Core component structure, form validation logic, mobile menu toggling, and IntersectionObserver scroll animations.

---

## 🚀 Getting Started

### 1. Install Dependencies
```bash
npm install
```

### 2. Start Development Server
```bash
npm run dev
```

### 3. Build for Production
```bash
npm run build
```

---

## 📂 Changing Site Information
To update the details of the company in the future, simply open [src/config.js](file:///c:/Users/adity/Downloads/vinay/src/config.js) and modify the values inside the `CONFIG` object. The website UI will automatically update based on the modifications.
